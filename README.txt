2x10W Stereo Speakers
=====================

See YouTube videos:

3D Printed Stereo Speaker Cases
For Eagle L018A 10W Speakers & STA540 Multiwatt Amplifier
https://www.youtube.com/watch?v=Qkf_tXnQIXA

STA540 Stereo & TDA7266M Mono Bridge Amplifiers Using A Common PCB

The project contains:

Multi Watt Amplifier/   - KiCad Circuit schematic & PCB Gerber files for 2x10W
                          Stereo bridge amplifiers using STA540 amplifier in a
                          MultiWatt package.

                          NOTE: CARE MUST BE TAKEN TO CONNECT THE POWER IN THE
                                CORRECT POLARITY UNLESS A POLARISED CONNECTOR
                                IS USED.

10W75mmCabinet/         - FreeCAD 3D Model Design & STL files for 3D Printing
                          the Eagle L018A 10W speaker cases.
